Verificador de Brechas de Seguridad

Script en Python que permite verificar si un correo electrónico ha sido comprometido en brechas de seguridad utilizando la API de Have I Been Pwned.

 📦 Requisitos
 Python 3.8 o superior
 API Key de Have I Been Pwned
