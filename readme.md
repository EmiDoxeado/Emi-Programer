Verificador de Correos con Have I Been Pwned Script que permite usar la Api de HIBP para analizar si un correo ha sido comprometido en algun momento.

Instalación Requiere la libreria "requests" de Python.

Uso Instalar "requests" Ejecutar el main.py (Sugerencia) Antes de usar, ya tener una ApiKey para ingresar al Script
