# Emi-Programer
Este proyecto contiene una colección de scripts desarrollados en PowerShell que permiten realizar tareas básicas de automatización.

Los scripts están enfocados en demostrar el uso de funciones, módulos y manejo del sistema de archivos, así como la interacción con el usuario mediante la consola.

El objetivo del proyecto es aprender a desarrollar herramientas simples que puedan reutilizarse en otros scripts o proyectos.
